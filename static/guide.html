Choosing Between Local Model and API Model

Local AI Model

Advantages:

  • Privacy: Your data stays on your local machine, ensuring privacy and security.
  • Totally Free: You won't need anything else than your device.

Disadvantages:

  • Resource Intensive: Requires significant computational resources and storage.
  • Experimental: The AI model is Gemini Nano which is used with the chrome experimental built-in AI which is still under development.
    Check out Chrome Documentation

API Model

Advantages:

  • Performance: No worrying about hardware limitations.

Disadvantages:

  • Data Privacy: Your data is sent to a remote server, in this case Google's server as Gemini is Google's latest AI model.
  • Also free: You will need a Gemini API key, which is free, you just have to login with your account at this link Google AI Studio.
